## 二、页面结构

### 2.1 整体布局（参考Penderecki's Garden沉浸式设计）

```
┌─────────────────────────────────────────────────────────────────┐
│                      Web端整体布局                               │
│                  （灵感来源：Penderecki's Garden）               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  桌面端布局（≥1024px）- 沉浸式全屏章节                           │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  [粒子3D背景 - 全屏WebGL点云可视化]                     │   │
│  │                                                         │   │
│  │  ┌───────────────────────────────────────────────────┐ │   │
│  │  │ 顶部悬浮导航栏 (透明/毛玻璃效果)                   │ │   │
│  │  │ [☰] 镜中                    [搜索]    [👤] [⚙]    │ │   │
│  │  └───────────────────────────────────────────────────┘ │   │
│  │                                                         │   │
│  │  ◄                                                    ► │   │
│  │  前                      当前章节                      后 │   │
│  │  进                                                    导 │   │
│  │  箭                      (100vh)                      航 │   │
│  │  头                                                    箭 │   │
│  │                                                     头   │   │
│  │  ┌─────────────────────────────────────────────────┐   │   │
│  │  │                                                 │   │   │
│  │  │              章节内容区域                        │   │   │
│  │  │              (对话/世界线/推演等)               │   │   │
│  │  │                                                 │   │   │
│  │  └─────────────────────────────────────────────────┘   │   │
│  │                                                         │   │
│  │  ┌───────────────────────────────────────────────────┐ │   │
│  │  │ 底部悬浮工具栏                                    │ │   │
│  │  │ [对话] [世界线] [画像] [记录]    进度指示器      │ │   │
│  │  └───────────────────────────────────────────────────┘ │   │
│  │                                                         │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  【核心设计原则】                                                │
│  ─────────────────────────────────────────────────────────────  │
│  · 全屏沉浸：每个主要功能区占据100vh                            │
│  · 粒子背景：WebGL点云持续漂浮，作为整个体验的视觉基调          │
│  · 透明导航：顶部和底部导航悬浮透明，不遮挡粒子背景             │
│  · 章节切换：左右箭头导航，类似Penderecki的章节切换             │

│  ─────────────────────────────────────────────────────────────  │

│  │  全屏沉浸体验，粒子背景占据整个视口                      │   │
│  │  内容叠加在粒子背景上，使用毛玻璃容器                    │   │
│  │  通过左右箭头切换不同章节                                │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │

│  ┌──────────────┬────────────────────────────────────────────┐ │
│  │  侧边栏      │              主内容区                       │ │
│  │  (可折叠)    │              (透明背景+毛玻璃卡片)          │ │
│  └──────────────┴────────────────────────────────────────────┘ │
│                                                                 │
│  响应式布局                                                     │
│  ─────────────────────────────────────────────────────────────  │
│  桌面端（≥1024px）：章节式/分栏可选                             │
│  平板端（768-1024px）：分栏收起为抽屉                           │
│  移动端Web（<768px）：纯章节式，底部Tab导航                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.4 页面优先级（Web实现顺序）

| 优先级 | 页面 | 说明 |
|-------|------|------|
| P0 | 开屏动画 | 与App一致：黑暗→点灯→人物→靠近→镜面→Slogan→问候→功能列表 |
| P0 | 全屏粒子背景 + 导航框架 | WebGL点云、透明导航、章节切换 |
| P0 | 自我探索章节 | 热力点探索、用户画像3D人体、毛玻璃对话容器 |
| P0 | 碎片记录章节 | 编辑器、粒子预览、热力点 |
| P0 | 平行推演章节 | 四格漫画、粒子画面、沉浸式体验 |
| P1 | 世界线章节 | 时间线热力点、节点详情弹窗 |
| P1 | 记忆注入章节 | 数据导入界面 |
| P1 | 全屏菜单覆盖层 | 章节导航、热力点入口 |
| P1 | 自我探索流程 | 冷启动体验（快速对话/详细探索选择） |
| P2 | 设置页 | 辅助功能 |

