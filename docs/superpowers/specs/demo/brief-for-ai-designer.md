# 镜中（Mirror）- Demo产品设计方案

> 本文档供AI设计师快速理解产品并设计页面交互逻辑
> 版本：v2.1
> 更新日期：2026-04-26

---

## 一、产品一句话

**镜中是一款通过长期对话理解用户，帮助用户推演"如果当初"和辅助当下决策的AI产品。**

---

## 二、Demo核心展示场景

Demo需要展示5个核心场景，让用户快速理解产品价值：

| 场景 | 用户操作 | 展示价值 |
|------|---------|---------|
| **冷启动** | 自我探索（完整版/快速版）+ 记忆注入 → 生成画像 | "哇，它懂我" |
| **碎片记录** | 记录日常片段 → 生成粒子图像 → 存档动效 | "我的时刻被珍藏了" |
| **平行推演** | 选择历史/未来 → 确认节点关系 → 观看动态漫画 | "原来那条路是这样的" |
| **决策分析** | 描述当下决策困境 → 获得客观分析 | "帮我理清思路" |
| **用户画像** | 查看粒子3D人体 → 了解自己的多维度 | "这是我吗" |

### 3.2 页面优先级（Demo实现顺序）

| 优先级 | 页面 | 说明 |
|-------|------|------|
| P0 | 主对话页 | 核心交互，展示日常使用场景 |
| P0 | 世界线页 | 展示记录成果，粒子封面节点 |
| P0 | 推演播放页 | 核心价值展示，动态漫画+粒子切换 |
| P0 | 碎片记录页 | 日常记录入口，粒子图像生成 |
| P1 | 画像生成页 | 冷启动体验，粒子汇聚动效 |
| P1 | 用户画像页 | 粒子3D人体展示 |
| P1 | 平行推演入口页 | 推演类型选择+节点关系确认 |
| P2 | 自我探索选择页 | 首次使用入口 |
| P2 | 记忆注入页 | 数据导入 |
| P2 | 设置页 | 辅助功能 |

## 九、粒子视觉技术要点

```
┌─────────────────────────────────────────────────────────────────┐
│                    粒子视觉技术规范                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  【技术栈】                                                      │
│  ─────────────────────────────────────────────────────────────  │
│  · 优先方案：WebGL + Three.js + Custom Shaders                 │
│  · 降级方案：Canvas 2D（低端设备）                               │
│  · 参考：Penderecki's Garden 粒子效果                           │
│                                                                 │
│  【核心着色器参数】                                              │
│  ─────────────────────────────────────────────────────────────  │
│  · uTime：时间因子，驱动动画                                    │
│  · uScale：粒子缩放 (0.5-2.0)                                  │
│  · uSize：基础粒子大小 (1.0-5.0)                               │
│  · uPositionRandom：位置随机度 (控制汇聚/分散)                 │
│  · uAlpha：透明度 (0.0-1.0)                                    │
│  · uMove：位移动画强度                                         │
│                                                                 │
│  【粒子密度配置】                                                │
│  ─────────────────────────────────────────────────────────────  │
│  · 开屏动画：10,000 粒子                                        │
│  · 用户画像3D人体：50,000 粒子                                  │
│  · 世界线节点封面：20,000-30,000 粒子                           │
│  · 动态漫画画面：15,000-25,000 粒子                             │
│  · 碎片记录背景：5,000-10,000 粒子                              │
│                                                                 │
│  【颜色系统】                                                    │
│  ─────────────────────────────────────────────────────────────  │
│  · 背景：#0A0A0A (纯黑)                                        │
│  · 粒子渐变：#E8D5B7 (暖黄) → #FAFAFA (白)                     │
│  · 强调色：#D4A574 (金色)                                       │
│                                                                 │
│  【动画参数】                                                    │
│  ─────────────────────────────────────────────────────────────  │
│  · 开屏汇聚：2-3秒，positionRandom 1.0→0.0                     │
│  · 节点展开：0.5秒，粒子分散后汇聚                              │
│  · 3D人体转动：持续，y轴旋转 0.1 rad/s                          │
│  · 漫画切换：0.8秒四幕过渡                                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
