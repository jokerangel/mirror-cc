## 一、设计风格

### 1.1 整体风格

```
┌─────────────────────────────────────────────────────────────────┐
│                        设计风格定调                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│  深沉、内省、温暖、治愈、私密                                    │

│  【情感基调】                                                    │
│  夜深人静时，一个人面对自己                                      │

### 8.4.3 动画参数配置

┌─────────────────────────────────────────────────────────────────┐
│                    动画参数配置                                  │
├─────────────────────────────────────────────────────────────────┤

│  ─────────────────────────────────────────────────────────────  │
│  // 顶点着色器中的位移计算                                       │
│  float noiseX = snoise(vec2(amplitude.x * 10.0, uTime * 0.1)); │

│  float noiseZ = snoise(vec2(amplitude.z * 10.0, uTime * 0.2)); │

│  displaced.x += amplitude.x * noiseX * uMove.x * depth;         │
│  displaced.y += amplitude.y * noiseY * uMove.y * depth;         │
│  displaced.z += amplitude.z * noiseZ * uMove.z * depth;         │
│                                                                 │
│  【动画参数建议值】                                              │
│  ─────────────────────────────────────────────────────────────  │
│  · uTime 速度：0.05-0.15（控制漂浮周期）                         │
